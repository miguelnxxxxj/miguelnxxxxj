## 📦 Projetos

### jClans

Plugin de clãs para Minecraft 1.21 com foco em performance e organização.

Sistemas presentes no plugin:

* API própria para desenvolvedores
* Ranking de clãs
* Sistema de upgrades
* Estatísticas de kills e KDR
* Banco de dados integrado
* Menus customizados

### jTerrenos

Plugin de terrenos para Minecraft com foco em proteção, gerenciamento e praticidade para os jogadores.

Sistemas presentes no plugin:

* Proteção de terrenos
* Sistema de permissões por jogador
* Menus customizados
* Configurações flexíveis
* Banco de dados integrado
* Sistema otimizado para performance

### VeltrixAPI

API que estou desenvolvendo com foco em organização, desempenho e facilidade de integração.

Objetivos do projeto:

* Estrutura moderna e escalável
* Integração simples para desenvolvedores
* Sistema otimizado para alta performance
* Recursos reutilizáveis para diferentes projetos
* Desenvolvimento contínuo com novas funcionalidades
